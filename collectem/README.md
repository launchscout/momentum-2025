<h1 align="center">Example GoLang component for Edgee</h1>

This is an example of a GoLang Edgee Component.

## Setup
Requirements:
- [edgee-cli](https://github.com/edgee-cloud/edgee)
- tinygo
- wasm-tools

```shell
$ make setup
```
## Building

```shell
$ make build
```
